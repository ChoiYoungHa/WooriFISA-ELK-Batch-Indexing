## 학습 목적 ⛷

- Crontab 활용
<br>

## 주요 기능
- 크롤링 후 DB 데이터 적재
- DB 데이터 Elastic Search 전달
- Crontab 자동화
<br>

## 사용 기술 스택
- 크롤링: 
- 데이터베이스: MySQL
- 검색 서비스: ELK Stack (Elasticsearch, Logstash, Kibana)
- 컨테이너화: Docker Compose
